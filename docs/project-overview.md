# Project Overview

## Purpose
Showcase portfolio projects and experience for Chasen Wang with a recruiter-friendly, code-inspired design.

## Highlights
- Single-page flow with smooth navigation.
- Data-driven content (projects/experience arrays).
- Hero section with react-three-fiber background.
- Mobile-ready layout via Tailwind.

## Tech Stack Summary
- Next.js 16, React 19, Tailwind 4, Framer Motion, three.js

## Architecture Type
- Monolithic frontend, static export.
